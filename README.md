# FPGA 8-bit Microprocessor (Verilog)

This project implements a simple 8-bit microprocessor using Verilog HDL and deploys it on an Artix-7 FPGA (Basys 3 board).

## Features

- 8-bit RISC architecture
- ALU supporting arithmetic and logical operations
- Register file
- Instruction decoder
- FSM-based control unit
- UART transmitter and receiver
- SPI controller

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Basys 3 FPGA board

## Architecture

The processor contains the following modules:

- ALU
- Control Unit
- Register File
- Instruction Decoder
- UART Interface
- SPI Controller

## Simulation

Processor functionality verified using Vivado simulation.

## Hardware Implementation

The design was synthesized and implemented on an Artix-7 FPGA using the Basys 3 development board.
